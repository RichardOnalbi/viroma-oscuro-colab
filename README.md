# viroma-oscuro-colab
Google Colab pipeline for neotropical Culicidae dark virome analysis
# Viroma oscuro de Culicidae neotropicales

Pipeline en Google Colab para organizar datos, tablas y figuras de un artículo de reflexión sobre el viroma oscuro de Culicidae neotropicales, con énfasis en Colombia, Brasil, análisis por homología y perspectivas con modelos de lenguaje proteico.

## Archivo principal

Abrir en Google Colab:

https://colab.research.google.com/github/RichardOnalbi/viroma-oscuro-colab/blob/main/pipeline_viroma_oscuro_google_colab.ipynb

## Contenido

- `pipeline_viroma_oscuro_google_colab.ipynb`: cuaderno principal de Google Colab.
- `pipeline_viroma_oscuro_google_colab.py`: versión script del pipeline.
- `viroma_oscuro_outputs/`: tablas, figuras y brief técnico generados.
- `README_pipeline_google_colab_viroma_oscuro.md`: guía detallada de uso.

## Propósito

El pipeline consolida un dataset curado de estudios neotropicales, genera tablas y figuras para el manuscrito, y deja preparado un módulo opcional para reprocesar datos SRA con Fastp, MEGAHIT, Prodigal y DIAMOND.
